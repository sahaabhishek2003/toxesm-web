About ToxESM

ToxESM is a peptide toxicity prediction web platform designed to provide accurate, scalable, and user-friendly analysis of peptide sequences. The system enables researchers to rapidly screen peptides and classify them as toxic or non-toxic using modern protein representation learning combined with robust machine learning approaches. It aims to bridge the gap between recent advances in protein language models and practical tools for biological research.

Predicting peptide toxicity is an essential step in areas such as therapeutic peptide design, drug discovery, and functional annotation. Experimental validation, while reliable, is often time-consuming and resource-intensive. Computational approaches therefore play a critical role in pre-screening peptide candidates before laboratory testing. However, many existing tools rely on handcrafted features or shallow sequence representations, which may fail to capture complex biological patterns.

ToxESM addresses this limitation by leveraging transformer-based protein language models to generate high-dimensional embeddings directly from raw peptide sequences. These embeddings capture contextual, structural, and evolutionary information that traditional feature engineering approaches often miss. By combining these rich representations with well-established machine learning models, the platform achieves a balance between predictive performance and computational efficiency.

The platform supports multiple machine learning models, including XGBoost, LightGBM, and Extra Trees, each selected based on consistent performance across evaluation metrics. In addition to individual models, ToxESM provides a conservative ensemble mode that prioritizes safety by reducing false negatives. In this mode, a peptide is classified as toxic if any model predicts toxicity, making it particularly suitable for high-sensitivity screening applications.

Beyond classification, ToxESM offers optional physicochemical property analysis, including molecular weight, isoelectric point, net charge, hydrophobicity (GRAVY), instability index, and aromaticity. These features provide additional biological context and allow users to better interpret the properties of predicted peptides without relying on external tools.

To ensure reliability, the models integrated within ToxESM have been evaluated on independent datasets and benchmarked against existing peptide toxicity prediction tools. The results demonstrate competitive performance across multiple metrics, including ROC-AUC, PR-AUC, F1-score, and MCC. This evaluation framework was designed to minimize bias and provide users with confidence in the predictions generated by the system.

ToxESM is designed with usability and accessibility in mind. It supports both single sequence input and batch processing through FASTA files, allowing efficient analysis of multiple peptides. The interface is intentionally simple and focused, enabling users to perform predictions without unnecessary complexity. Results are presented in a structured format and can be downloaded for further analysis.

Overall, ToxESM provides a modern, integrated solution for peptide toxicity prediction, combining advances in protein language modeling with interpretable machine learning. The platform emphasizes accuracy, transparency, and ease of use, making it a valuable resource for researchers in bioinformatics, computational biology, and related fields.
