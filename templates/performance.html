7. Model Interpretability (SHAP Analysis)

To interpret model predictions, SHAP (SHapley Additive exPlanations) analysis was performed on the XGBoost model. SHAP values quantify the contribution of each embedding feature toward the final prediction.

The SHAP summary plot shows the distribution of feature impacts across all samples. Features with higher SHAP values contribute more strongly toward toxicity prediction, while lower values push predictions toward non-toxic classification.

The bar plot highlights the most influential embedding dimensions based on their average contribution to model output. A small subset of features consistently dominates the prediction process, indicating that the model relies on key latent representations learned by ESM-2.

Overall, the interpretability analysis confirms that the model captures meaningful sequence-level patterns and utilizes them effectively for toxicity classification. Combined with strong performance metrics, this demonstrates that the ToxESM framework is both accurate and reliable for peptide toxicity prediction.

The combined evaluation and interpretability results demonstrate that transformer-based embeddings integrated with tree-based models provide a powerful framework for peptide toxicity prediction. Among the evaluated models, XGBoost achieved the most consistent performance, while SHAP analysis confirmed that predictions are driven by a focused set of informative embedding features.
