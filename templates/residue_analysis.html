Residue Analysis

This module provides residue-level interpretability for peptide toxicity prediction. Each amino acid is evaluated to determine its contribution to the predicted toxicity.

The analysis uses an occlusion-based approach, where each residue is systematically perturbed and the resulting change in prediction score is measured.
